This project was used as a reminder course. Since only blueprints were used, it was easy to follow up and re-learn the basics. The course link can be found here. The things that I learnt with this course are following ones :
  • Basic character and object movements
  • Level Design
  • Basic Game Design
  • Event dispatches, Blueprint Interfaces
  • Cinematics
  • Creating in-game menu
  • 2D Game Development; Sprites, Textures etc.
